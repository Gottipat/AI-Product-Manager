# Context Files (Antigravity)

Shared context for both team members using Antigravity.

## Files

| File        | Purpose          | Update Frequency      |
| ----------- | ---------------- | --------------------- |
| `CLAUDE.md` | Project overview | Rarely                |
| `DOMAIN.md` | Domain knowledge | When adding concepts  |
| `API.md`    | API reference    | When adding endpoints |
| `SPRINT.md` | Current work     | **Daily**             |

## Team Sync

Keep `SPRINT.md` updated so Antigravity knows:

- What you're working on
- What's blocked
- What's completed

This ensures consistent AI assistance for both team members.
